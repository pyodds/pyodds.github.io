automl.cash module¶

class automl.cash.CASH(name: str = 'CASH', data: pandas.core.frame.DataFrame = None, ground_truth: pandas.core.frame.DataFrame = None, )¶

Combined Algorithm Selection and Hyperparameter search (CASH) is an Hyperparameter Optimization technique that can simultaneously search for the best algorithm and the best hyperparmeter configuration for that algorithm within its objective function

Parameters

  • name (str, optional (default='AutoEncoder')) – The name of the algorithm

  • data (pandas.core.frame.DataFrame, (default=None)) – The dataset which must be studied on

  • ground_truth (pandas.core.frame.DataFrame, (default=None)) – The labels for the dataset

objective_function(param: dict) → numpy.float64¶

Takes a point in the Search Space, and returns its corresponding loss (quantitative measure of failure). This method, converts the search point into its corresponding algorithm, trains the algorithm on the train split, and records its failure on the test split. This quantitative measure of failure on test split is returned as loss :param dict: A deterministic point in the AutoML Search Space

Returns

loss – Quantitative measure of model failure. The objective is to minimize this measure

Return type

numpy.float64

model_selector(max_evals: int) → algo.base.Base¶

Predict raw anomaly score of X using the fitted detector. The anomaly score of an input sample is computed based on different detector algorithms. For consistency, outliers are assigned with larger anomaly scores. :param max_evals: The number of iterations of the Search algorithm before termination
